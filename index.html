🇷🇺 создаёт первый код (Offer). 🇦🇲 вставляет его и получает ответный код (Answer). Потом 🇷🇺 вставляет Answer — и вы на связи.

Как пользоваться
  1. Оба нажимают “Включить микрофон (и камеру)”.
  2. Оба выбирают свою сторону: 🇷🇺 или 🇦🇲.
  3. 🇷🇺 получает “Ваш код” → отправляет 🇦🇲.
  4. 🇦🇲 вставляет код → жмёт “Вставил(а) → продолжить” → получает “Ваш код” → отправляет 🇷🇺.
  5. 🇷🇺 вставляет ответный код → жмёт “Вставил(а) → продолжить”.

Важно: коды могут быть длинными — это нормально. В некоторых сетях прямое соединение может не установиться без TURN-сервера.
